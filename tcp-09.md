# 滑动窗口：发送窗口与接收窗口

### 滑动窗口：发送窗口快照

1. 已发送并收到 Ack 确认的数据：1-31 字节
2. 已发送未收到 Ack 确认的数据：32-45 字节
3. 未发送但总大小在接收方处理范围内：46-51 字节
4. 未发送但总大小超出接收方处理范围：52-字节

### 可用窗口/发送窗口

* 可用窗口：46-51 字节 / 发送窗口：32-51 字节

### 46-51 字节已发送

* 可用窗口耗尽

### 32 到 36 字节已确认

* 发送窗口移动

### 发送窗口

1. 已发 送已接收
2. 已发送未接收
3. 未发 送且在发送窗口内
4. 未发送且在发送窗口外

### 约等于对端发送窗口的接收窗口
* RCV.WND
* RCV.NXT
