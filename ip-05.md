# NAT 地址转换与 LVS 负载均衡

NAT（IP Network Address Translator）
LVS（Linux Virtual Server）

### NAT 应用的前提
* 内网中主要用于客户端访问互联网
* 同一时间仅少量主机访问互联网
* 内网中存在一个路由器负责访问外网

### NAT 优点

* 优点
  - 共享公共 IP 地址，节约开支
  - 扩展主机时不涉及公共地址
  - 更换 ISP 服务商（更换公网 IP 地 址），不对主机地址产生影响
  - 更好的安全性，外部服务无法主动 访问内网服务
  - 更好的隔离性
* 缺点
  - 网络管理复杂
  - 性能下降
  - 重新修改校验和
  - 客户端缺乏公网 IP 导致功能缺失
  - 某些应用层协议由于传递网络层信 息而功能受限
